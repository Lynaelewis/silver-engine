#Extended linked lists
This project builds on a basic linked list in Python by adding several useful variants. The goal was to practice object-oriented design, data structure customization, and algorithm efficiency.
Features
- ReversableLinkedList – A linked list that can be reversed in place. Perfect for understanding pointer manipulation and O(n) algorithms.
- SortedLinkedList – Keeps all elements in ascending order automatically when you insert them. Prevents unordered additions and demonstrates sorted insertion logic.
- UniqueLinkedList – Removes duplicate elements while tracking how many times each item occurred. Great for practicing hash-based tracking in data structures.
